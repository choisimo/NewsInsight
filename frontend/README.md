# News Sentiment Analysis

키워드 기반 NewInsight 및 인사이트 제공 플랫폼

## Project Overview

This project provides real-time news sentiment analysis based on keywords, visualizing positive/negative sentiment and key insights.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Chart.js & Recharts for data visualization

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server with hot-reload
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## Features

- Real-time news sentiment analysis
- Interactive keyword cloud visualization
- Sentiment trend charts
- Responsive design with modern UI components
